# 1. HTML - Introducción, Sintaxis y Estructura

## 📚 Objetivos de Aprendizaje

- Comprender qué es HTML y su importancia en la web

- Conocer las etiquetas más comunes y su uso
- Crear documentos HTML bien formados

## 🎯 Conceptos Clave

### ¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Define la estructura y contenido de las páginas mediante etiquetas.

### Estructura Básica de HTML

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la Página</title>
</head>
<body>
    <!-- Contenido aquí -->
</body>
</html>
```

### Etiquetas Principales

| Etiqueta | Descripción | Ejemplo |
|----------|-------------|----------|
| `<h1>` a `<h6>` | Encabezados | `<h1>Título Principal</h1>` |
| `<p>` | Párrafo | `<p>Texto de párrafo</p>` |
| `<a>` | Enlace | `<a href="url">Enlace</a>` |
| `<img>` | Imagen | `<img src="imagen.jpg" alt="Descripción">` |
| `<div>` | Contenedor | `<div>Contenido</div>` |
| `<ul>`, `<ol>` | Listas | `<ul><li>Item</li></ul>` |
| `<table>` | Tabla | `<table><tr><td>Celda</td></tr></table>` |

## 📝 Reglas de Sintaxis

1. **Etiquetas**: Siempre cierran `<etiqueta>contenido</etiqueta>`
2. **Atributos**: Se escriben en la etiqueta de apertura `<img src="" alt="">`
3. **Indentación**: Usa espacios o tabulaciones para organizar
4. **Anidación**: Las etiquetas deben estar correctamente anidadas
5. **DOCTYPE**: Siempre al inicio del documento
