🚀 JavaScript - Variables y Tipos de Datos

Explora los conceptos fundamentales de JavaScript interactivamente.

📝 Variables

Las variables son contenedores para almacenar valores.

let nombre = "Juan"; // String let edad = 25; // Number const ciudad = "Madrid"; // Constante

🎨 Tipos de Datos

JavaScript tiene varios tipos de datos primitivos:

// String (Texto) let texto = "Hola Mundo"; // Number (Números) let entero = 42; let decimal = 3.14; // Boolean (Verdadero/Falso) let esActivo = true; // Array (Lista) let colores = ['rojo', 'verde', 'azul']; // Object (Objeto) let persona = { nombre: 'Juan', edad: 25 };

🔢 Operadores Aritméticos

Operaciones matemáticas básicas:

let a = 10; let b = 3; console.log(a + b); // 13 - Suma console.log(a - b); // 7 - Resta console.log(a * b); // 30 - Multiplicación console.log(a / b); // 3.33... - División console.log(a % b); // 1 - Módulo console.log(a ** b); // 1000 - Potencia

⚖️ Operadores de Comparación

Comparar valores y devuelven true o false:

let x = 5; let y = "5"; console.log(x == y); // true (comparación de valor) console.log(x === y); // false (comparación estricta) console.log(x != y); // false console.log(x > 3); // true console.log(x < 10); // true

📚 Operaciones con Strings

Manipulación de texto:

let nombre = "Juan"; let apellido = "Pérez"; // Concatenación let nombreCompleto = nombre + " " + apellido; // Template Literals let saludo = `Hola, ${nombre}!`; // Métodos console.log(nombre.length); // 4 console.log(nombre.toUpperCase()); // JUAN console.log(nombre.toLowerCase()); // juan

📋 Arrays (Listas)

Trabajo con arrays:

let numeros = [1, 2, 3, 4, 5]; // Acceder a elementos console.log(numeros[0]); // 1 // Longitud console.log(numeros.length); // 5 // Añadir elemento numeros.push(6); // Iterar numeros.forEach(num => console.log(num));

🔧 Funciones

Crear y usar funciones:

// Función tradicional function saludar(nombre) { return `Hola, ${nombre}!`; } // Arrow Function const sumar = (a, b) => a + b; console.log(saludar("Juan")); console.log(sumar(5, 3)); // 8
