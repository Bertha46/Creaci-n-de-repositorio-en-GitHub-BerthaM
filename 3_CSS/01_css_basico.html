🎨 Introducción a CSS

Aprende selectores y propiedades CSS fundamentales

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje de estilos que se utiliza para definir cómo se presentan los elementos HTML. Permite controlar aspectos visuales como colores, tipografía, espaciado y diseño.

Características Principales

🎯 Selectores

Elementos, clases, IDs y más para aplicar estilos específicos

🌈 Colores

Define colores de texto, fondo y bordes de forma flexible

📏 Espaciado

Controla márgenes, relleno y tamaños con precisión

Tipos de Selectores

Tipo Sintaxis Ejemplo
Elemento p p { color: blue; }
Clase .nombre .titulo { font-size: 24px; }
ID #nombre #principal { width: 100%; }
Atributo [atributo] input[type="text"] { padding: 5px; }

Propiedades CSS Esenciales

  • color: Define el color del texto
  • background-color: Color de fondo
  • font-size: Tamaño de la fuente
  • padding: Espacio interior
  • margin: Espacio exterior
  • border: Borde del elemento
  • display: Tipo de visualización (block, inline, flex, grid)
  • width/height: Ancho y alto del elemento
