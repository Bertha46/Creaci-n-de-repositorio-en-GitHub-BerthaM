# Repositorio de Trabajos de la Asignatura

¡Bienvenido! Este repositorio almacena todos los trabajos y proyectos desarrollados durante la asignatura.

## 📋 Descripción

Este repositorio ha sido creado para centralizar y organizar todos los trabajos prácticos, proyectos y ejercicios realizados a lo largo del curso.

## 🎯 Objetivos

- Mantener un registro ordenado de todos los trabajos desarrollados
- Practicar control de versiones con Git y GitHub
- Colaborar y compartir conocimientos

## 📁 Estructura del Repositorio

```
.
├── README.md          # Este archivo
├── trabajos/          # Carpeta para trabajos individuales
├── proyectos/         # Carpeta para proyectos principales
└── recursos/          # Carpeta para materiales de referencia
```

## 🚀 Cómo Usar Este Repositorio

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/Bertha46/Creaci-n-de-repositorio-en-GitHub-BerthaM.git
   ```

2. **Crear una rama para tu trabajo:**
   ```bash
   git checkout -b nombre-del-trabajo
   ```

3. **Hacer cambios y commit:**
   ```bash
   git add .
   git commit -m "Descripción clara del cambio"
   ```

4. **Subir cambios:**
   ```bash
   git push origin nombre-del-trabajo
   ```

## 📝 Requisitos de Commits

- Usa mensajes claros y descriptivos
- Agrupa cambios relacionados en un solo commit
- Usa el presente: "Agregar feature" en lugar de "Agregué feature"
